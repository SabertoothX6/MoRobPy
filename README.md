# Value Iteration
This is a quick and dirty try to implement Value Iteration in a grid world written in Python

## Classes

### Gridworld
This class is used to create and store the grid world. It also contains the functions to calculate the Value Iteration

### Printer
Printer is used to print the grid world and values to the console

### Reader
Reads in a JSON-file and creates a grid world from it
